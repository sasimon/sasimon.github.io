Welcome to Templatizer!

This form will apply a template to your existing Web page code as is required for the Webpage Design and Development courses at Tulane University taught by Dr. George North. It uses JavaScript in its code that invokes the jQuery library. The HTML code is generated on submission of the form (by pressing the Templatize button). The CSS code is for the top, "header" comment of the CSS file only, and relies only on the current date. It is generated on page load.
